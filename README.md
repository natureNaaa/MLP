# 流动手帐 · 2023-2026

一个单页面的手绘涂鸦风格网站，自动播放 2023–2026 年间的迁徙旅程。时间推进时，地图会平移，小人物沿着手绘路径行走，节点处弹出便签式的文字叙事。

## 预览

直接在浏览器打开 `index.html` 即可查看动画，无需额外构建步骤。

如果需要一个可分享的访问链接，可在项目根目录运行轻量本地服务器：

```bash
./serve.sh
```

默认监听 `http://0.0.0.0:3000`，便于在容器或远程环境暴露端口后直接访问。如需变更端口/主机，设置环境变量即可：

```bash
HOST=127.0.0.1 PORT=8080 ./serve.sh
```

随后在浏览器访问对应地址（如 <http://localhost:3000>），或将服务器暴露到公网后分享该 URL。

- 地图范围覆盖中国与欧洲，并以低饱和、略带抖动的涂鸦线条呈现。
- 轨迹会被逐步绘制，小人物始终停留在画面中央附近，带有轻微的晃动感。
- 每个迁徙节点附带时间、地点与 2–4 行便签式描述。

## 文件说明

- `index.html`：页面结构与 SVG 地图定义。
- `style.css`：纸张纹理、手帐配色、人物与便签样式。
- `script.js`：时间驱动的行走动画、地图平移与节点便签控制。
